# 🎬 Alist视频刷一刷  

**为什么做这个项目？**  
你是否也想过——Alist上有海量视频资源，却缺少一个像短视频平台那样「点击即看」的轻量网页？尽管网上现有方案不多（或许觉得这功能太简单），不过我自己觉得很好玩，我决定自己动手！于是，这个网页诞生了✨  

**特点直击**  
- 🚀 **极简交互**：点一点切换视频，无需复杂操作  
- 📁 **无缝对接Alist**：直接读取你的存储挂载资源  
- 🎨 **自适应界面**：手机/电脑都能流畅使用  
- 🔄 **即装即用**：静态页面直开
